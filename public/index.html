For publishers · Lumi SDK

Monetize your AI app in one line of code.

Drop in the Lumi SDK and turn free users into revenue without ruining UX. Native, contextual placements that fit inside AI chats, agents, and assistants — no banners, no surveillance.

  • ✓Native ad formats — built for conversations, not billboards.
  • ✓Real-time payouts — track impressions, clicks, and earnings live.
  • ✓Full creative control — decide what, when, and where ads appear.
Integrate the SDK → 
// Add Boost Boss in one line
import { BoostBoss } from '@boostbossai/lumi-sdk';

const bb = new BoostBoss({
  apiKey: 'bb_sk_live_***',
  format: 'native'
});

const ad = await bb.fetch({
  context: userMessage,
  audience: 'ai-native'
});

render(ad); // 💰
