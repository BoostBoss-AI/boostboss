Open API

Programmatic access to your share links, clicks, conversions, and payouts. Build MCP servers that serve Boost Boss products, automate link minting from your CMS, or pipe conversion data into your own analytics stack.

My API

Secret — (apply below)

API access is currently invite-only while we tune rate limits. Email api@boostboss.ai to request access — typical turnaround 1 business day. API keys are scoped to your affiliate account only (read-your-own-data); no cross-affiliate access.

Quickstart

Boost Boss exposes a REST API at https://api.boostboss.ai/v1. All endpoints accept the App ID + Secret as Bearer credentials and return JSON. Rate limit: 60 requests/minute per key.

Example — list your share links + click counts:

curl https://api.boostboss.ai/v1/share-links \
  -H "Authorization: Bearer $BB_API_SECRET" \
  -H "X-BB-App-ID: $BB_APP_ID"

# Response
{
  "links": [
    { "token": "aBcDeFgH", "url": "https://boostboss.ai/s/aBcDeFgH",
      "product": "Stripe Atlas", "clicks": 42, "last_click_at": "2026-06-12T15:00:00Z" }
  ],
  "total_clicks": 42
}
